## 요약

원하는 층과 회의실을 선택하고 달력에서 날짜를 더블클릭한 후, 시간과 내용을 입력하면
예약이 진행된다. 단, '입력' 버튼을 누른 뒤에도 사용 매너 관련 확인 팝업을 한 번 더
거쳐야 예약이 최종 완료된다. 예약 내용 수정도 같은 방식으로 열어 간단하게 가능하다.

---

## 1. 회의실 예약 방법

원하는 회의실을 선택하고 날짜·시간·내용을 입력하여 예약하는 방법과, 예약 내용을 수정하는 방법을 설명한다.

### 1.1. 회의실 예약 절차

1. **구분 및 회의실 선택**
   1. 좌측 상단 '구분' 영역에서 층(11층본부 / 13층본부 / 자기층 중 선택)을 선택한다.
   2. 아래 활성화된 회의실 체크박스 목록(예: A1~A4, B1~B4)에서 예약할 회의실을 체크한다.
2. **예약 정보 입력**
   1. 중앙 달력에서 원하는 날짜를 더블클릭하면 '회의실 예약' 팝업이 열린다.
   2. 팝업 내 '시작시간'/'종료시간'의 시·분을 드롭다운 메뉴에서 선택한다.
   3. '회의내용' 입력 필드에 회의 내용을 입력한다.
   4. 팝업 우측 상단의 '입력' 버튼을 눌러 예약을 진행한다.
3. **주의사항 확인 (완료 조건)**
   1. '회의실 예약 확인' 안내 팝업(사용 매너 관련 주의사항 1~4번, 예: "미사용 시 즉시 취소",
      "스크린/화이트보드 정리")이 뜬다.
   2. 이 팝업을 확인·승인해야 예약이 최종 완료된다 — '입력' 버튼 클릭만으로는 완료되지 않는다.

### 1.2. 예약 내용 확인 및 수정

1. **예약 내용 확인**
   1. 예약 완료 시, 또는 우측 '조회' 버튼을 누르면 입력된 예약 내용을 조회하여 확인할 수 있다.
      달력의 해당 날짜·시간 영역에 노란색 블록으로 표시된다.
   2. 다른 층을 선택하면 해당 층의 예약 내용이 달력에 실시간으로 전환되어 보여진다.
2. **예약 내용 수정**
   1. 수정하고 싶은 예약 블록(달력 상)을 더블클릭하면 상세 팝업이 열린다.
   2. '회의내용' 필드의 기존 텍스트를 지우고 새 내용을 입력한다.
   3. 팝업 우측 상단의 '수정' 버튼을 누르면 수정이 완료되고, 달력에서 최종 확인할 수 있다.

## 2. 화면 UI 요소 참고

- **화면번호**: TR3691 (회의실 예약 화면)
- **좌측 패널**: '구분'(11층본부/13층본부/자기층), '회의실'(A1~A4, B1~B4 등 체크박스), 연/월 이동 버튼
- **중앙 패널**: 달력(월~일)
- **우측 패널**: '예약자'/'회의내용'/'회의실명' 표시 필드, '조회'/'입력'/'수정'/'제거' 버튼
- **예약/수정 팝업**: '회의일자', '예약시간'(시작/종료 드롭다운), '회의내용' 입력 필드, '입력'/'수정'/'제거'/'닫기' 버튼
- **안내 팝업**: '회의실 예약 확인' 타이틀 + 사용 매너 주의사항 1~4번
